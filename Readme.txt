1) Download weights file from this link https://pjreddie.com/media/files/yolov3.weights
2) Add weights file in the yolo folder
3) Download Project
4) open project with jupyter notebook

- To add a test image upload it in yolo folder 

1) then Go to Test On image Section in the Object-Detection notebook
2) change name of the image in the img_path

- To add a test Video upload it in project folder

2) change the name according to your desired input video
3) change the output video name according to your desire
